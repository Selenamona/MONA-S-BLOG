## | 日期格式转换

```javascript
Vue.prototype.$formatDate = time => {
  if (time && time != '') {
      let dataTime = new Date(time).toLocaleDateString().replace(/\//g, "-");
      dataTime = dataTime.split('-').map(item => {
          if (item > 0 && item < 10) {
              return '0' + item;
          } else {
              return item;
          }
      }).join('-');
      return dataTime;
  } else {
      return ''
  }
}
```
