`(10)["toString"]() === "10"`  // true

> 参考：[js奇淫技巧1](http://qylanikin.lofter.com/post/1cbb3f55_cc787c9) 

`null == undefined`    // true  
`null === undefined`   // false   
`"abc" === "abc"`           // true   
`1 === 1`       // true  
`[1,2,3] === [1,2,3]`       // false  
`{a: 1} === {a: 1}`         // false  
`{} === {}`                 // false   

`Math.min() < Math.max()`   // false 【Math.min() 返回 Infinity, 而 Math.max()返回 -Infinity】
