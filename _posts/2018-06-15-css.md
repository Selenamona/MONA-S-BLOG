### 单行省略
```css
{
  overflow: hidden;   
  text-overflow:ellipsis;   
  white-space: nowrap;   
}
```

### 多行省略

```css
{
  overflow:hidden;
  text-overflow:ellipsis;
  display:-webkit-box;
  -webkit-line-clamp:2;
  -webkit-box-orient:vertical;
}
{
  /*  黑科技==解决box-orient失效； 用注释先关闭 autoprefixer 然后再开启；*/
  /* autoprefixer: off */  
  -webkit-box-orient: vertical;  
  /* autoprefixer: on */  
}
```
