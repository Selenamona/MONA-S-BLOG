1. 电脑安装证书/添加信任

    帮助 -> SSL代理 -> 安装 charles 根证书[当前用户/授信任的根证书办法机构]

2. 手机端安装证书/添加信任

    帮助 -> SSL代理 -> 在移动设备或远程浏览器上安装  charles 根证书

    手机设置代理Ip和端口号，浏览器打开指定页面安装证书。IOS 关于本机 -> 证书信任设置

3. charles -> 代理 -> SSL代理设置 -> 勾选“启用SSL代理” -> 客户端证书 -> 添加主机和证书
